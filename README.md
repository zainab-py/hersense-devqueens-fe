# hersense-devqueens-fe
HerSense is a women-focused health and safety application designed to support everyday decisions through simple, contextual insights.

The platform allows users to log menstrual cycle data, mood, and symptoms, which are then used to provide more relevant responses instead of generic suggestions. It focuses on practical use cases that people deal with daily rather than long-term tracking alone.

A key feature is ingredient-based analysis for skincare and haircare products. Users can check what they are using and understand how specific ingredients may affect their skin or scalp, especially in relation to current conditions like dryness or sensitivity.

The app also includes context-aware suggestions such as sunscreen guidance based on timing and exposure, making recommendations more relevant instead of repetitive notifications.

For safety, HerSense integrates emergency support with nearby hospital mapping and an SOS feature that can share live location with trusted contacts. Medical assistance features such as nearby doctors and appointment support are also included.

To maintain reliability, the system only suggests home remedies and avoids providing medication advice.

The project is built using Loveable for the frontend, Supabase for backend and authentication, and the Gemini API for handling user interaction and contextual responses.

HerSense is designed to be simple, usable, and responsive to real situations—something that can be relied on regularly, not just tracked occasionally.
